# Importante
El dataset pesa demasiado y no se puede añadir a git. Para añadirlo seguid los pasos:
1. Descargar la version full de 2025 del [dataset](https://www.kaggle.com/datasets/artermiloff/steam-games-dataset/data?select=games_march2025_full.csv).
2. Renombrarlo a `kaggle_march2025_full.csv`.
3. Moverlo a la carpeta _/data/_.
