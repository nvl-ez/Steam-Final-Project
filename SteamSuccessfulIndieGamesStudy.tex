\section{Introduction}\label{introduction}

\subsection{Explanation of the
Attributes}\label{explanation-of-the-attributes}

The dataset we will be working with contains a total number of 94948
observations and 47 columns or variables. The columns that will be used
are described bellow:

\begin{itemize}
\tightlist
\item
  \textbf{\texttt{appid}}: Unique idetnifier of the game on Steam.
  {[}num{]}
\item
  \textbf{\texttt{name}}: Name of the game. {[}text{]}
\item
  \textbf{`released\_date``}: Represents the date where the game was
  released. {[}time{]}
\item
  \textbf{`required\_age``}: Corresponds to the minimum age required to
  play the game. {[}num{]}
\item
  \textbf{\texttt{price}}: How much the game costs. If its 0 it means
  that the game is Free to Play. {[}num{]}
\item
  \textbf{\texttt{dlc\_count}}: Ammount of DLCs (Downloadable Contents)
  the game has. {[}num{]}
\item
  \textbf{\texttt{suppport\_url}}: URL to the support page of the game.
  {[}text{]}
\item
  \textbf{\texttt{windows}}: Determines if the game runs in windows.
  {[}categorical{]}
\item
  \textbf{\texttt{mac}}: Determines if the game runs in mac.
  {[}categorical{]}
\item
  \textbf{\texttt{linux}}: Determines if the game runs in linux.
  {[}categorical{]}
\item
  \textbf{\texttt{metacritic\_score}}: Metacritic score based on
  critcical reviews (reviews from professionalss). By performing an
  investigation we think that an score of 0 means that when the scraping
  of the data was done there where no reviews for that game yet.
  {[}num{]}
\item
  \textbf{\texttt{achievements}}: Number of achievements the game has.
  {[}num{]}
\item
  \textbf{`recommendations``}: Ammount of user recommendations.
  {[}num{]}
\item
  \textbf{\texttt{supported\_languages}}: List of languages that the
  game supports.{[}\textbf{NO SÉ QUE CATEGORÍA DARLE}{]}
\item
  \textbf{\texttt{packages}}: Avaliable packages for the game. It
  contains the name and a description of the package and the names,
  descriptions and subprices of the subpackages. {[}\textbf{LO MISMO QUE
  ARRIBA}{]}
\item
  \textbf{\texttt{Developers}}: List of developers associated with the
  game. {[}\textbf{LO MISMO QUE ARRIBA}{]}
\item
  \textbf{\texttt{publishers}}: List of publishers associated with the
  game. {[}\textbf{LO MISMO QUE ARRIBA}{]}
\item
  \textbf{\texttt{categories}}: List of categories that the game has.
  {[}\textbf{LO MISMO QUE ARRIBA}{]}
\item
  \textbf{\texttt{genres}}: List of genres that the game belongs to.
  {[}\textbf{LO MISMO QUE ARRIBA}{]}
\item
  \textbf{\texttt{positive}}: Ammount of positive votes the game has.
  {[}num{]}
\item
  \textbf{\texttt{negative}}: Ammount of negative votes the game has.
  {[}num{]}
\item
  \textbf{\texttt{estimated\_owners}}: Estimated owners of the game.
  {[}text{]}
\item
  \textbf{\texttt{average\_playtime\_forever}}: Average playtime since
  March 2009 measured in minutes. {[}num{]}
\item
  \textbf{\texttt{average\_playtime\_2weeks}}: Average playtime in the
  last two weeks measured in minutes. {[}num{]}
\item
  \textbf{\texttt{median\_playtime\_forever}}: Median playtime since
  March 2009 measured in minutes. {[}num{]}
\item
  \textbf{\texttt{median\_playtime\_2weeks}}: Median playtime in the
  last two weeks measured in minutes.{[}num{]}
\item
  \textbf{\texttt{peak\_ccu}}: Number of current users playing the day
  before the data was scrapped. {[}num{]}
\item
  \textbf{\texttt{tags}}: List of tags the game has with its name and
  its key. {[}\textbf{NO SE QUE CATEGORIA DARLE}{]}
\item
  \textbf{\texttt{pct\_pos\_total}}: Percentage of all reviews that are
  positive. {[}num{]}
\item
  \textbf{\texttt{num\_reviews\_total}}: Nummber of the total reviews
  the game has. {[}num{]}
\end{itemize}
